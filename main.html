Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in the DOM . First of all we can use Single Element Selection example: getElementById and querySelector. Again we can used Multiple Elements Selection example:querySelectorAll and getElementsByName. we can used Traversing and Advanced Selection.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

the difference between innerHTML, innerText, and textContent are innerHTML retrieves or sets the HTML content inside an element, including tags.And the secoend part is innerText returns only the visible text content, ignoring hidden elements and respecting CSS styling. And the third is textContent returns all text content, including hidden elements, without interpreting HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to manage events for all its child elements.Also This approach leverages event bubbling, allowing you to handle events efficiently and especially for dynamically added or numerous child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates upward through its ancestor elements in the hierarchy. This allows parent elements to handle events that originate from their children, enabling techniques like event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement. To add it to the DOM, use methods like appendChild or insertBefore, and to remove an element, use removeChild or remove.
